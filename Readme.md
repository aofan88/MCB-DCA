一、項目概述

MCB 自動掃描系統是一套由 Joe Chan 自主研發、部署於專屬 VPS 的 美股自動化監控與買入信號捕捉系統，基於 Market Cipher B 指標深度客製化而成。

本系統每日依據美股最新收盤資料，自動分析 50 檔 S&P500 大型權值股，並即時推送結果至 Telegram。

其核心目標是：

用嚴謹、數學化的方式，捕捉「極度超賣→反轉」的黃金買點，並自動通知投資者，達到量化化、系統化、無情緒的交易紀律。

這套系統目前已全自動化運作，並可無限擴充。

⸻

二、策略核心邏輯（MCB-Joe 版）

本策略基於 Market Cipher B 的 WT 波段（wt1 與 wt2）指標進行優化與數學化處理，完全對應 TradingView 自製指標版本，達到信號一致。

1. 超賣反轉核心

透過 EMA、偏差、平滑技術計算 WT1（青波）、WT2（黃波），並抓取：
	•	波段交叉（cross）
	•	綠點反轉信號（green dot）
	•	深度超賣條件（wt1 < -35 / -60）

2. 兩級買入信號

⭐ Buy1（1x 倉位）
	•	綠點成立
	•	且 wt1 < -35
→ 輕度超賣反轉

🔥 Buy2（2x 倉位）
	•	綠點成立
	•	且 wt1 < -60
→ 極度超賣、極易出現強反彈

本策略屬於 超賣反轉 / 低風險切入 類型，適合波段與長線選股。

⸻

三、系統架構與技術特點

本系統構建於自家 VPS，搭配 EODHD 金融資料 API，並以 Python 建立完整自動化流程。

1. 資料來源
	•	EODHD API（付費級別資料）
直接取得美股日線官方級別資料，避免 Yahoo Finance 的多種錯誤與延遲。

2. 計算引擎
	•	使用 Pandas + NumPy
	•	完整重現 TradingView MCB 計算邏輯
	•	自動計算 wt1、wt2、綠點、1x/2x 信號

3. 自動化排程
	•	VPS 設定每日自動執行
	•	台北時間上午 09:00 固定推送
	•	無須人工干預

4. 通知與人機交互
	•	Telegram Bot 自動推送訊息
	•	即使無信號也會推送「Heartbeat / 心跳包」
	•	保證系統正常運作
